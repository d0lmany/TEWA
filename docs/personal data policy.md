# Политика обработки персональных данных

Вся информация хранится постоянно и удаляется немедленно после удаления аккаунта.

### 1.1. Все пользователи
| Данные | Тип | Цель сбора | Шифрование |
|--------|-----|------------|------------|
| **Обязательные** | | | | |
| Email | string | Аутентификация |  AES-256 |
| Имя | string | Оформление заказов, доставка |  - |
| Пароль (hash) | string | Безопасность |  bcrypt |
| Дата рождения | date | Персонализация, безопасность |  - |
| **Опциональные** | | | | |
| Аватар | file | Профиль |  - |
| Адреса доставки | json | Оформление заказов, доставка |  - |
