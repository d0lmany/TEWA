**TEWA** (Template E-commerce Web Application) - это шаблон SPA онлайн-магазина/маркетплейса, созданный для быстрого запуска интернет-бизнеса без долгих настроек, как в традиционных CMS.

> *Проект может служить основой для реальных коммерческих решений, но автор не несёт ответственность за возможные риски*

## Основные возможности
- Просмотр товаров, их подробностей и отзывов
- Поиск и фильтрация товаров
- Регистрация и авторизация пользователей
- Функциональные списки избранного
- Функциональная корзина
- Полный список функций приведён в [этом файле](./functions.md), следите за обновлениями в [CHANGELOG](./CHANGELOG.md)

## Технологический стек
### Frontend
- **Vue.js 3** - *Прогрессивный фреймворк JavaScript. Доступный, производительный и универсальный фреймворк для создания пользовательских веб-интерфейсов*
- **Vue Router** - *Маршрутизатор Vue. Выразительная, настраиваемая и удобная маршрутизация для Vue.js*
- **Pinia** - *Интуитивно понятное хранилище для Vue.js*
- **Element Plus** - *Библиотека компонентов на базе Vue 3 для дизайнеров и разработчиков*
- **TypeScript** - *Строго типизированный язык программирования, основанный на JavaScript, предоставляющий вам лучшие инструменты в любом масштабе*

### Backend
- **Laravel 12** - *это PHP-фреймворк для веб-приложений с выразительным, элегантным синтаксисом и надежной экосистемой*
- **PHP 8.4** - *Популярный скриптовый язык общего назначения, который особенно подходит для веб-разработки*

### Инструменты разработки
- **phpMyAdmin** - *Бесплатный программный инструмент, написанный на PHP, предназначенный для администрирования MySQL через Веб*
- **VSCode** - *Редактор кода c ИИ с открытым исходным кодом*
- **Drawio** - *Построение схем для команд, ориентированных в первую очередь на безопасность*
- **Bruno** - *Заново изобретаем API-клиент. Git-интегрированный, полностью автономный API-клиент с открытым исходным кодом*

## Установка и запуск
### Требования
- PHP 8.4+
- Node.js 16+
- Composer
- MySQL 8+/MariaDB 10+

### Быстрый старт
```bash
# 1. Клонируем репозиторий и переходим в него
git clone https://github.com/d0lmany/TEWA
cd tewa

# 2. Устанавливаем зависимости
# для фронтенда
cd frontend 
npm install 
# для бэкенда
cd ../backend
composer install

# 3. Создаём БД
# в вашей СУБД создайте БД с именем 'tewa' (для более тонкой настройки обратитесь к файлу `./backend/.env`)
# запускаем миграции
php artisan migrate

# 4. Запускаем приложение в dev mode
cd ..
./run # запустит сервера для бэкенда и фронтенда, отключить с помощью Ctrl+C
```
